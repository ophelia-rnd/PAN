# Parallel Anomalous Nudge (PAN) algorithm

_Parallel Anomalous Nudge (PAN)_ is a semi-supervised machine learning model that detects anomalies by measuring and combining deviations from historical normal and abnormal Self-Organizing Map representations. The model is being developed as a [scikit-learn](https://scikit-learn.org/stable/)-compatible algorithm.

## Documentation

Installation documentation, API documentation, and examples can be found in the [documentation](https://ophelia-rnd.github.io/PAN).
