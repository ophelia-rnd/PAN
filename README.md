# Parallel Anomalous Nudge (PAN) algorithm

_Parallel Anomalous Nudge (PAN)_ is a semi-supervised machine learning model that detects anomalies by measuring and combining deviations from historical normal and abnormal Self-Organizing Map representations.

This work is guided by the following objectives:

* **Research:** Identify background and gaps, Formalize proposed solution
* **Experiments:** Design experiments, Demonstrate (visually) the proposed method, Evaluation and benchmarking
* **Development:** Implement the proposed method as an `Sklearn`-compatible estimator
